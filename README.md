# Dev Bootstrapper

A CLI tool designed to **automate project setup** by generating standardized directory structures and boilerplate files.

## Features
- Quickly scaffold new projects
- Enforces consistent project structure
- Reduces manual setup time
- Designed for extensibility (future templates)

## Tech Stack
- Python
- argparse

## Usage
```bash
python bootstrapper_app.py -c python -n new_project                                                     
```
